# Package index

## All functions

- [`add_fix_id()`](https://spatialmarine.github.io/intersimR/reference/add_fix_id.md)
  : Add a per-track sequential fix index and a globally-unique fix
  identifier
- [`as_track()`](https://spatialmarine.github.io/intersimR/reference/as_track.md)
  : Coerce inputs to a canonical movement track table
- [`create_oceanmask()`](https://spatialmarine.github.io/intersimR/reference/create_oceanmask.md)
  : Create a land/ocean mask raster for track simulations
- [`detect_proximity_events()`](https://spatialmarine.github.io/intersimR/reference/detect_proximity_events.md)
  : Detect proximity-based events between animal and vessel
- [`extract_event_segments()`](https://spatialmarine.github.io/intersimR/reference/extract_event_segments.md)
  : Extract animal and vessel segments (track_tbl) for a selected
  proximity event
- [`is_track_tbl()`](https://spatialmarine.github.io/intersimR/reference/is_track_tbl.md)
  : Test if an object is a track table
- [`make_point_check()`](https://spatialmarine.github.io/intersimR/reference/make_point_check.md)
  : Create a point.check function for availability::surrogateAR()
- [`match_animal_vessels()`](https://spatialmarine.github.io/intersimR/reference/match_animal_vessels.md)
  : Match animal fixes to vessel positions within time and distance
  thresholds
- [`print(`*`<track_tbl>`*`)`](https://spatialmarine.github.io/intersimR/reference/print.track_tbl.md)
  : Print method for track tables
- [`simulate_tracks()`](https://spatialmarine.github.io/intersimR/reference/simulate_tracks.md)
  : Simulate surrogate tracks for an animal segment
- [`test_interactions()`](https://spatialmarine.github.io/intersimR/reference/test_interactions.md)
  : Test animal-vessel interactions (attraction / follow) using
  simulations
- [`timegap_segments()`](https://spatialmarine.github.io/intersimR/reference/timegap_segments.md)
  : Segment a sorted time series by a maximum gap (in minutes)
